# RTB 2.0 Complete Rewrite Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rewrite RTB from Node.js/TypeScript to Rust, implementing a hybrid remote terminal + AI Agent management platform with plugin architecture.

**Architecture:** Rust Cargo workspace with 5 crates (core, server, plugin-host, cli, tauri-app). Event-driven architecture using tokio::broadcast for control events and per-session mpsc for data events. React 19 + Vite + Tailwind frontend embedded via rust-embed.

**Tech Stack:** Rust 1.82+ (Axum, tokio, portable-pty, dashmap), React 19, Vite 6, Tailwind CSS 4, shadcn/ui, xterm.js 6, Tauri 2, clap 4

**Spec:** `docs/superpowers/specs/2026-03-25-rtb-2.0-redesign.md`
